# Django Starter

## Virtual Environment

Create Environment
```
py -m venv myworld
```

Activate Environment
```
{folder}\Scripts\activate.bat
```

## Install Django
```
pip install django
```

## Start Project
```
django-admin startproject projectname
```

## Create App
```
python manage.py startapp appname
```

## Migration
General Migration
```
python manage.py migrate
```
Model Make Migration
```
py manage.py makemigrations {appname}
```
